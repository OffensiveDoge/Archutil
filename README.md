# Archutil
A basic bash package installation script for Arch Linux

Bugs still may exist, but all should be fine

Current Packages include:
 - AUR
    - Yay
    - Paru
 - Browser
    - Brave
    - Firefox
    - Librewolf
    - Mullvad
 - Terminal
    - Konsole
    - Terminator
    - Tilix
    - Kitty
 - File Manager
    - Dolphin
    - Thunar
    - Yazi
  - Text Editor
    - Kate
    - Nano
    - Vim
    - Neovim
 - Libreoffice
 - Fastfetch
 - Btop
 - Kcalc
 - GIMP
 - VLC
 - 7zip
 - Timeshift
 - Minecraft Launcher
